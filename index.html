Bookmark boilerplate


javascript:(function(){
	var el = document.createElement('script');
	el.src = 'https://your-site.com/bookmarklet.js';
	document.getElementsByTagName('body')[0].appendChild(el);
})();

This is basically it

  • A bookmarklet is basically JS in a bookmark URL.
  • You can inject a JS script into the page
  • Gotchas: Content Security Policy directive (CSP)

The question is what to add into it

Navigation, User and
Resource Timing API

Some of the most usefull tools in the bookmarklet or perf tool toolkit

Navigation Timing API

http://www.w3.org/TR/navigation-timing

  • No explicit redirect data for Cross domain redirect

Examples

performance.timing


var totalRequest = performance.timing.domComplete - performance.timing.navigationStart;
var connectionTime = performance.timing.responseStart - performance.timing.navigationStart;

User Timing API


performance.mark("startTask");

setTimeout(function(){
	performance.mark("endTask");

	//measure elapsed time
	performance.measure("TaskTotal", "startTask", "endTask");

	//output duration
	console.log(performance.getEntriesByName("TaskTotal").duration);
}, 500);

http://www.w3.org/TR/user-timing

  • e.g. to emasure XHR requests or code

Resource Timing API

http://www.w3.org/TR/resource-timing

  • Similar to Navigation timing but for assetes
  • Gotcha blocking

Examples


var resources = performance.getEntriesByType("resource");
console.table(resources.map(function(resource){
    return {
	    "name" : resource.name,
	    "total" : Math.round(resource.duration) + "ms"
    }
}));
Logs a sortable table with duration of each resource
